dnovel
------------
dnovel 是一个简洁，干净的小说聚合系统

## 概述

[后端地址](https://github.com/dreamlu/dnovel)

[前端flutter地址](https://github.com/dreamlu/dnovel-flutter)

长期维护,学习colly自用  

### 界面  
![界面](static/img/dnovel_flutter.jpeg)

### 部署  
- docker部署  
进入docker目录,执行update.sh即可  
- 源码  
go build  

默认端口为 8022
